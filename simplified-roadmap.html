Расписание День за Днём

Структурированный 12-недельный путь к освоению фулстек-разработки

Неделя 1: Основы HTML и CSS

День 1-2: Введение в HTML

Начните свой путь в веб-разработке с изучения HTML - основного строительного блока веб-сайтов.

Темы:
  • Структура HTML документа и элементы
  • HTML5 семантические теги
  • Текстовое форматирование и списки
  • Ссылки и изображения
  • Таблицы и формы
  • Валидация HTML
Практический проект:

Создайте базовую HTML-страницу с биографией, включающую заголовки, параграфы, списки, изображения и ссылки.

Ресурсы для обучения:
  • MDN: Введение в HTML
  • W3Schools HTML учебник
  • FreeCodeCamp: Basic HTML

День 3-4: Основы CSS

Изучите, как стилизовать ваши HTML-страницы с помощью CSS, превращая их из простого текста в визуально привлекательные веб-страницы.

Темы:
  • Подключение CSS к HTML
  • Селекторы и специфичность
  • Цвета, фоны и границы
  • Блочная модель (margin, padding, border)
  • Типографика и стилизация текста
  • Свойства отображения и позиционирование
Практический проект:

Применить стили CSS к вашей HTML-странице с биографией, добавляя цвета, шрифты, границы и оформление для создания привлекательного дизайна.

Ресурсы для обучения:
  • MDN: Первые шаги в CSS
  • CSS-Tricks: Руководства
  • FreeCodeCamp: Basic CSS

День 5-7: CSS Макеты и Flexbox

Освойте современные методы создания макетов с помощью CSS, фокусируясь на Flexbox для построения адаптивных, гибких дизайнов.

Темы:
  • Принципы макетов CSS
  • Введение в Flexbox
  • Flexbox-контейнеры и flex-элементы
  • Выравнивание и распределение пространства
  • Вложенные flex-контейнеры
  • Создание общих шаблонов макетов
Практический проект:

Создайте навигационную панель и структуру макета с помощью Flexbox для вашей страницы биографии, добавляя заголовок, боковую панель и раздел основного контента.

Ресурсы для обучения:
  • MDN: Flexbox
  • CSS-Tricks: Полное руководство по Flexbox
  • Flexbox Froggy (игра)

Неделя 2: Основы JavaScript

День 8-9: Введение в JavaScript

Начните изучение JavaScript, языка программирования, который добавляет интерактивность вашим веб-страницам.

Темы:
  • Основы JavaScript и подключение к HTML
  • Переменные, типы данных и операторы
  • Условные операторы (if/else, switch)
  • Функции и область видимости
  • Базовый вывод на консоль и отладка
  • Работа с математическими операциями
Практический проект:

Создайте простой калькулятор с пользовательским интерфейсом, который может выполнять базовые арифметические операции.

Ресурсы для обучения:
  • MDN: Первые шаги с JavaScript
  • W3Schools JavaScript руководство
  • JavaScript.info: Основы JavaScript

День 10-11: JavaScript Циклы и Массивы

Изучите, как работать с коллекциями данных и циклами для обработки множества значений и структурированных данных.

Темы:
  • Циклы for, while, и do-while
